

Mel Psicología - Sistema de Gestión de Citas
Este es un sistema web integral diseñado para el consultorio de Mel Psicología. El proyecto se aleja de los diseños saturados para ofrecer una experiencia Japandi (Zen-Minimalista), utilizando una paleta de colores salvia y arena que transmite calma, confianza y profesionalismo tanto a pacientes como al administrador.

Características Principales
Experiencia Zen para el Paciente: Interfaz limpia con efecto Glassmorphism y tipografía suave, ideal para reducir la ansiedad antes de una consulta.

Gestión Administrativa Simplificada: Panel de control intuitivo para publicar, visualizar y eliminar espacios de citas en tiempo real.

Diseño Responsivo & Moderno: Optimizado con la fuente Outfit y micro-interacciones suaves que funcionan de maravilla en móviles y escritorio.

Seguridad: Acceso protegido para el administrador con manejo de sesiones y validación de credenciales.

Stack Tecnológico
Backend: Java (Servlets & JSP)

Servidor: Eclipse GlassFish 6.2.1

Base de Datos: MySQL (XAMPP)

Frontend: HTML5, CSS3 (Custom Properties & Flexbox), JavaScript Vanilla

Tipografía: Google Fonts (Outfit)
